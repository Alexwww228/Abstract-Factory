# Банковская система с шаблоном Abstract Factory

Это **консольное приложение на C#**, демонстрирующее использование **шаблона проектирования Abstract Factory** для обслуживания различных типов клиентов: **Retail**, **Corporate** и **VIP**.

## Описание

Программа имитирует работу банка, обслуживающего три типа клиентов:

1. **Retail** — обычные клиенты
2. **Corporate** — корпоративные клиенты
3. **VIP** — премиум-клиенты

Каждый тип клиента имеет свой **набор продуктов**:

* Счёт (`IAccount`)
* Кредит (`ILoan`)
* Банковская карта (`IBankCard`)

Для создания этих продуктов используется **Абстрактная Фабрика** (`IBankFactory`), что позволяет клиентскому коду работать с целым семейством продуктов **без знания их конкретных классов**.

---

## Функции

* Открытие и управление банковскими счетами
* Пополнение и снятие средств
* Одобрение кредитов и расчёт процентов
* Выпуск банковских карт и совершение платежей
* Лёгкое расширение: добавление новых типов клиентов или продуктов

---

## Архитектура

* **IBankFactory** — создаёт семейство продуктов
* **Конкретные фабрики** — создают продукты для конкретного типа клиента
* **Интерфейсы продуктов** — задают методы для работы со счетами, кредитами и картами
* **BankService** — клиентская часть, использует фабрику для обслуживания клиентов

---

## Классы

| Класс                                                         | Назначение                                              |
| ------------------------------------------------------------- | ------------------------------------------------------- |
| `IAccount`                                                    | Интерфейс банковского счёта                             |
| `ILoan`                                                       | Интерфейс кредита                                       |
| `IBankCard`                                                   | Интерфейс банковской карты                              |
| `RetailAccount`, `CorporateAccount`, `VipAccount`             | Конкретные реализации счетов для разных клиентов        |
| `RetailLoan`, `CorporateLoan`, `VipLoan`                      | Конкретные кредиты для разных клиентов                  |
| `RetailCard`, `CorporateCard`, `VipCard`                      | Конкретные карты для разных клиентов                    |
| `IBankFactory`                                                | Абстрактная фабрика                                     |
| `RetailBankFactory`, `CorporateBankFactory`, `VipBankFactory` | Конкретные фабрики для разных типов клиентов            |
| `BankService`                                                 | Клиентская часть, обслуживает клиента с помощью фабрики |
| `Program`                                                     | Основная программа, консольный интерфейс                |
